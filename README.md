# HireShield

Hackathon Chrome extension for on-device deepfake detection during live interview calls.

## Repo layout

- [`hireshield-extension/`](/Users/omererturk/Documents/Coding/HireShield/hireshield-extension) - load this folder as the unpacked Chrome extension

## Quick start

1. Open `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select `hireshield-extension`

For detailed extension usage, demo mode, and model notes, see [`hireshield-extension/README.md`](/Users/omererturk/Documents/Coding/HireShield/hireshield-extension/README.md).
